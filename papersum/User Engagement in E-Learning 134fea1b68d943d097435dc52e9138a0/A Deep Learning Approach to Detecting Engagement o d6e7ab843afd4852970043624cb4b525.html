What?

A deep learning-based approach to detecting online learnersâ€™ engagement through using their facial expressions. Two-level (not-engaged and engaged) and three-level (not-engaged, normally-engaged and veryengaged) decisions are made on engagement detection during classification.

A Local Directional Pattern (LDP) to extract person-independent edge features for the different facial expressions and Kernel Principal Component Analysis (KPCA) to capture the nonlinear correlations among the extracted features.

Why?

Performance Improvement over the dataset.

How?

A. Face Representation Using LDP

Once the histogram is obtained, Kernel Principal Component Analysis (KPCA) is applied for dimensionality reduction. In KPCA, a Gaussian kernel is applied on input features to transform them into a higher-dimensional feature space. Then, a typical PCA is applied on the high-dimensional space.

DBN-based Engagement Classification

The classification module employs deep belief network (DBN) for the engagement classification using the features obtained from KPCA. DBNs are found to be efficient in facial expression recognition applications, since pixel-based classification is not needed in such applications.

Results

The proposed method shows high accuracy in engagement classification. The experiment conducted on DAiSEE demonstrated that the proposed method achieves a high accuracy both in two-level (90.89%) and three-level (87.25%) engagement detections.
